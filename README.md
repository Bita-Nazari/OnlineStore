# Online Shop System

This is a web-based Online Shop platform designed with three levels of user roles: Admin, Seller, and Buyer.

## Features

### Admin
- Full access to the entire system
- Manage sellers, buyers, products, and all shop content
- Monitor transactions and platform activities

### Seller
- Create and manage multiple shop booths (stores)
- Add and manage products
- Create auctions for products to sell items via bidding system

### Buyer
- Browse sellers and products
- Add products to shopping cart
- Purchase items through checkout system
- Leave comments and reviews on products and sellers

## Project Overview

This platform allows sellers to independently manage their own shops while buyers can explore a wide range of products and participate in both direct purchases and auctions. The admin oversees the entire ecosystem to ensure smooth operation and control.

A multi-role marketplace system where sellers operate independently, buyers interact freely, and the admin maintains full control over the platform.
